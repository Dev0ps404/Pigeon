# ✨ Overview

**Pigeon** is a next-generation real-time communication platform built using the **MERN Stack** with advanced modern features such as:

* ⚡ Real-time messaging
* 👥 Group chats
* 📞 Audio & video calling
* 📂 Media & file sharing
* 🔔 Live notifications
* 🌙 Dark/Light mode
* 🤖 AI-powered features
* 🔒 Secure JWT authentication
* ☁️ Cloudinary uploads
* 📱 Fully responsive startup-level UI

Designed with a premium UI/UX inspired by:

* Discord
* Telegram
* Slack
* WhatsApp Web
* Linear
* Notion

# ⚙️ Installation

# 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/pigeon.git
cd pigeon
```

---

# 2️⃣ Install Dependencies

## Frontend

```bash
cd client
npm install
```

## Backend

```bash
cd ../server
npm install
```

---



# 3️⃣ Run Application

## Backend

```bash
cd server
npm run dev
```

## Frontend

```bash
cd client
npm run dev
```
